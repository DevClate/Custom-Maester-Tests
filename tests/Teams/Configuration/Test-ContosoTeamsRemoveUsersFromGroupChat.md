This test reviews whether chat owners can delete messages in group chats.

This informational test examines the message deletion permissions for chat owners, which is important for chat governance and moderation capabilities.

**To modify this setting:**

- Navigate to the [Teams admin center > Messaging policies](https://admin.teams.microsoft.com/policies/messaging)
- Select the Global (Org-wide default) policy
- Review the **Owners can delete sent messages** setting
- Enable if chat owners should have moderation capabilities
- Click **Save**

- [Manage messaging policies in Teams](https://learn.microsoft.com/en-us/microsoftteams/messaging-policies-in-teams)
