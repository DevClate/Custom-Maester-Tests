This test reviews whether people search is scoped to specific groups.

Scoped people search restricts directory searches to defined groups rather than the entire organization. This informational test reviews the setting for privacy and directory access considerations.

**To modify this setting:**

- Navigate to the [Teams admin center > Teams settings](https://admin.teams.microsoft.com/company-wide-settings/teams-settings)
- Scroll to the **Search by name** section
- Review the **Scope directory search using an Exchange address book policy** setting
- Enable if users should only search within their defined scope
- Click **Save**

**Learn more:**

- [Manage Teams settings for your organization](https://learn.microsoft.com/en-us/microsoftteams/enable-features-office-365)
- [Manage address book policies in Exchange Online](https://learn.microsoft.com/en-us/exchange/address-books/address-book-policies/address-book-policies)
