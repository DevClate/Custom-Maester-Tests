This test checks if URL previews are disabled in the Teams messaging policy.

Contoso's company policy requires URL previews to be disabled as they can leak information to external sites when links are shared in conversations. This is important for preventing metadata exposure and maintaining data security.

**To remediate this issue:**

- Navigate to the [Teams admin center > Messaging policies](https://admin.teams.microsoft.com/policies/messaging)
- Select the Global (Org-wide default) policy
- Set **URL previews** to **Off**
- Click **Save**

**Learn more:**

- [Manage messaging policies in Teams](https://learn.microsoft.com/en-us/microsoftteams/messaging-policies-in-teams)
- [Teams security best practices](https://learn.microsoft.com/en-us/microsoftteams/teams-security-guide)
