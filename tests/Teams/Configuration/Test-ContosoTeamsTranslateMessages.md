This test reviews message translation settings for sensitive data concerns.

Message translation features send message content to external translation services. This test reviews the setting to ensure organizations consider whether sensitive data could be exposed through third-party translation services.

**To modify this setting:**

- Navigate to the [Teams admin center > Messaging policies](https://admin.teams.microsoft.com/policies/messaging)
- Select the Global (Org-wide default) policy
- Review the **Translate messages** setting
- Consider disabling if sensitive data exposure is a concern
- Click **Save**

**Learn more:**

- [Manage messaging policies in Teams](https://learn.microsoft.com/en-us/microsoftteams/messaging-policies-in-teams)
- [Translate a message in Teams](https://support.microsoft.com/en-us/office/translate-a-message-in-teams-d8926ce9-d6a6-47df-a416-f1adb62d3194)
