This test checks if security end user reporting is enabled in Teams.

Contoso's company policy requires that users can report security concerns directly from Teams. This is important for maintaining a secure collaboration environment and enabling quick response to potential security threats.

**To remediate this issue:**

- Navigate to the [Teams admin center > Messaging policies](https://admin.teams.microsoft.com/policies/messaging)
- Select the Global (Org-wide default) policy
- Set **Allow security end user reporting** to **On**
- Click **Save**

**Learn more:**

- [Manage messaging policies in Teams](https://learn.microsoft.com/en-us/microsoftteams/messaging-policies-in-teams)
- [Report a concern in Microsoft Teams](https://learn.microsoft.com/en-us/microsoftteams/report-security-concern-in-teams)
